数据类型

JavaScript变量的数据类型是根据赋值的类型确定的

  1. Number(数字型，包括小数，默认0，Infinitu无穷大，-Infinity无穷小，NaN代表非数值)
  2. Boolean(布尔类型，等价于1/0，默认false)
  3. String(字符串类型，默认””)
  4. Undefined(无确认类型，产生原因：声明变量没有赋值)
  5. Null(空值，默认null)
  6. object()

isNaN方法：判断是否是非数字类型，如果是数字类型返回false；

运算符

运算符规则和用法和Java一样，有区别的是逻辑运算符有新的运算

逻辑中断：

  1. 与中断(表达式1 && 表达式2，如果表达式1为false，那么返回表达式1，如果表达式1为true，返回表达式2)
  2. 或中断(表达式1 || 表达式2，如果表达式1为false，那么返回表达式2，如果表达式1为true，返回表达式1)

数组

创建数组

  1. var arr=new Array();
  2. var arr=[内容];

JavaScript数组的特别：数组的内容是随意的，不像Java有特定的类型，比如var arr=[1，”我”]是一个合法数组

其余特点和Java的数组特点差不多

函数

格式：

function 方法名(形参){
JavaScript语句
}

特点：JavaScript函数的形参也不需要要特定类型，只需要桉顺序传参，第一个实参给第一个形参，实参什么类型形参就是什么类型。如果实参数量多于形参，那么忽略多余的实参。

arguments的使用：

  1. arguments存储了所有传递的实参
  2. 可以理解成数组，通过数组的调用方法调用arguments存储的实参
  3. arguments变成了形参，比如arguments[0]就是第一个实参的形参

作用域

和Java一样，每个变量都有作用域，可以分为全局变量和局部变量。和Java不同的是，JavaScript没有块级变量(es6除外)

块级作用域：在循环，选择语句中创建的变量，外部是不可以使用的，但JavaScript可以使用

预解析

js引擎运行js分两个步骤：预解析和代码执行

预解析：js引擎会把js代码里面所有的var和function放到他们作用域的最前面
代码执行：从上到下执行

案例1：

console.log(num)
var num=10

这段代码啊相当于：

var=num;
console.log(num)
num=10

输出结果为undefined

案例2：

fun()
var fun =fuction(){console.log(22)}

这段代码相当于：

var fun
fun()
fun =fuction(){console.log(22)}

输出结果：报错

案例3：

fun()
fuction fun (){console.log(22)}

这段代码相当于：

fuction fun (){console.log(22)}
fun()

输出结果：22

对象

创建对象的方法：

方法一(字面量创建对象)：

var obj ={
name=”me”,
number=1,
sex=”boy”,
saygood: function(){
console.log(“good”);
}
}

  1. 多个属性或者方法用逗号隔开
  2. 方法的创建是方法名：匿名函数
  3. 调用对象属性或者方法 我们用 对象名.属性名/方法名(参数)

方法二(new Object创建对象)：

var me = new Object();
me.name=”me”;
me.number=1;
me. sex=”boy”;
me.saygood= function(){
console.log(“good”);
}

  1. 这种方法和new Array类似
  2. 创建空对象，对对象进行属性和方法的添加
  3. 每个属性方法用分号隔开
  4. 调用对象属性或者方法 我们用 对象名.属性名/方法名(参数)

方法三()：

function Man (name，number，sex){
this.name=name;
this.number=number;
this.sex=sex;
this.saygood= function(){
console.log(“good”);
}
}

var zy=new Man(“me”,1,”boy”)

  1. 函数名首字母必须大写
  2. 调用构造函数需要用new
  3. this必不可少

遍历对象：for in方法

案例：

for(var key in zy){
console.log(key);这个key调用的是属性
console.log(zy[key]);这个zy[key]调用的是属性值
}

Math

  1. Math.PI圆周率
  2. Math.max()求最大值
  3. Math.floor()向小约等于
  4. Math.ceil()向大约等于
  5. Math.round()四舍五入
  6. Math.abs()绝对值
  7. Math.random()获取一个0~1的随机数

放大我们想要的随机整数：

  1. 选择范围
  2. min=Math.ceil(左范围)获取最小范围整数
  3. max=Math.floor(右范围)获取最大范围整数
  4. Math floor(Math.random()*(max-min+1))+min 利用此公式获取不小于min和不大于max的整数

Date

Date必须创建实例来使用：比如var date=new Date()

Date的使用：

  1. getFullyear()年
  2. getMonth()月(1月0~12月11)
  3. getDate()日
  4. getDay()星期几(周日0~周六6)
  5. getHours()时
  6. getMinutes()分
  7. getSecond()秒

小技巧：如果想让分时秒变成两位数(01样式)那就在前面加上’0’

Date获得毫秒数(1970.1.1到如今(默认到如今)的毫秒数)

  1. valueOf()/getTime()
  2. 创建实例的时候直接var date=+new Date()
  3. Date.now()

倒计时：输入的时间的毫秒数-如今的时间的毫秒数再转化为分时秒

Array

检测是否为数组：

  1. instanceof+要检测的对象
  2. Array.isArray(参数)

添加删除数组元素：

  1. push(参数1，参数2.。。)末尾添加n个元素
  2. pop()删除数组最后一个元素
  3. unshift(参数1，参数2.。。)数组开头添加n个元素
  4. shift()删除数组开头元素

点到数组顺序：reverse()

数组排序：sort([compareFunction])

  1. 默认排序：compareFunction不写
  2. 从大到小：compareFunction写function(a,b){return a-b(b-a就是小到大)}

获取数组元素下标：

  1. indexOf(要查询的元素，开始处(默认0))–只返回第一个匹配的元素下标
  2. lastIndexOf(要查询的元素)

数组转换为字符串：

  1. toString()把数组转换为字符串，逗号分隔每一段
  2. join(‘分隔符’)把数组所有元素转换为一个字符串

String

字符出现的位置

  1. indexOf
  2. lastIndexOf

根据下标返回字符

  1. charAt(下标)返回字符串该位置的字符
  2. charCodeAt(下标)返回字符串该位置字符串的ASCII码
  3. str[下标]H5新增方法

寻找字符串出现最多的字符：根据JavaScript对象特有的性质，创建空对象，字符串中出现的字符可以作为属性添加到对象中，遇到一次就添加属性并赋值1，遇到两次就是直接赋值2

连接截取字符串

  1. concat(str1,str2….)拼接字符串
  2. substr(start,length)截取length长度，从start开始
  3. slice/subString(start,end)截取从start开始，end结束不取end
