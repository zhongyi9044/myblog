获取元素

  1. document.getElementById(‘id名’)–获取有id的元素对象
  2. document.getElementsByTagName(‘标签名’)–返回带有指定标签名的对象集合
  3. document.getElementsByClassName(‘类名’)–根据类名返回元素对象集合
  4. document.querySelector(‘选择器’)–根据指定选择器返回第一个元素对象
  5. document.body返回body元素对象
  6. document.documentElement返回html元素对象

事件基础

事件三要素：

  1. 事件源
  2. 事件类型
  3. 事件处理程序

例子：点击按钮弹出对话框(事件源：按钮，事件类型：鼠标经过，事件处理程序：通过函数赋值实现)

var btn=document.getElement('id'); btn.onclick(事件类型)=function(){JavaScript语句}

执行事件过程：

  1. 获取事件源(获取元素)
  2. 绑定事件，注册事件(例如：btn.onclick(onclick就是一种事件类型，btn则是获取的元素))
  3. 添加处理事件程序

操作元素

修改元素内容：

  1. element.innerText
  2. element.innerHTML

二者区别：Text是不识别html标签(把HTML标签当作文本看待)并且不要空格和换行，HTML识别html标签同时保留空格和换行

常用元素的属性操作：

  1. src
  2. href
  3. id
  4. alt
  5. title
  6. type
  7. value
  8. checked
  9. selected
  10. disabled(是否可用，true可用)

步骤：

  1. 获取事件源元素和需要被操作的元素
  2. 注册事件源元素事件
  3. 对被操作元素进行处理程序

例子：点击按钮切换图片

  1. 获取按钮和图片元素
  2. 注册按钮事件
  3. 进行处理图片img的src程序

注意：事件函数内部如果需要更改注册事件的元素，那么可以使用this.的方式

修改样式属性：

  1. element.style行内样式操作(写在HTML内部的样式)
  2. element.className类名样式操作

使用例子：

  1. this.style.backgroundColor="red"注意：backgroundColor是驼峰命名法
  2. dv.className="change" change是一个新的样式，即把获取的元素dv的样式换成change类的

自定义属性(可以用于自定义)：

  1. element.setAttribute(‘属性’，’值’)设置属性(H5中，自定义属性开头都用data-xx)
  2. element.getAttribute(‘属性’)获取属性值
  3. element.removeAttribute(‘属性’)删除属性
  4. element.dataset.xxx(H5获取自定义属性,比如：date-index-number，获取：element.dataset.indexNumber)

  2. 鼠标点击，图片进行切换

节点操作

  1. element.parentNode父节点
  2. element.children所有子节点元素
  3. element.next/previousElementSibling上一个/下一个兄弟节点
  4. element.firstElementChild第一个子元素子节点
  5. element.lastElementChild最后一个子元素子节点

创建添加节点：

  1. document.createElement(‘tagName’)创建节点
  2. element.appendChild(child)添加节点(element是父级，child是子级，添加在父节点的末尾)
  3. element.insertBefore(child,指定元素)–(添加到父节点中指定节点的前面)

删除节点：

element.removeChild(child)–从父元素中删除一个子节点

克隆节点：

element.cloneNode(false/true)–false只复制标签，true复制标签和内容

事件高级

方法监听注册事件

同一个元素只能有一种注册事件，但可以有多个监听器

格式：eventTarget.addEvenListener(type,listener,useCapture)–这种注册事件方法是最好的

  1. type事件类型的字符串，比如click，mousover，注意这里没有‘on’
  2. listener事件处理函数，事件发生则会调用该函数
  3. useCapture可选参数，默认false，和事件流有关

删除事件

传统格式：注册事件=null(比如div.onclick=null)

方法监听格式：evenTarget.removeEventListener(type,listener,useCapture)写在listener函数内部

注意：因为删除事件是写在listener内部，所以我们需要把listener的函数写在外部，再由注册事件的时候调用此函数，且调用函数不需要加括号

DOM事件流

事件发生时，会在元素节点之间按照也定的顺序传播

  1. 捕获阶段
  2. 冒泡阶段
  3. 当前目标阶段

如何理解DOM的事件流：首先理解捕获阶段是从浅到深，冒泡阶段是深到浅，而JavaScript中代码只能执行捕获或者冒泡，且每个事件可能事件流不同

事件对象

eventTarget.addEvenListener(type,listener(e),useCapture)–e(可以自己随便命名)就是事件对象，不是参数，我们不需要传参。他是我们事件的相关数据的集合

事件对象常见属性和方法(写在listener内)：

  1. e.target返回触发事件的对象(和this的区别在于：this是绑定的元素，target是触发事件的元素，假如绑定了div，但内部有个li，点击li触发了div事件(冒泡)，那么target就是li)
  2. e.type返回事件类型，不带‘on’
  3. e.preventDefault()阻止默认行为，比如链接跳转
  4. e.stopPropagation()阻止事件冒泡

一些e.preventDefault()的用处：

  1. 事件类型为contextmenu时e.preventDefault()可以禁用右键菜单
  2. 事件类型为selectstart时e.preventDefault()可以阻止选中文字

BOM

BOM:浏览器对象模型，把浏览器看作一个对象，顶级对象是window，以上学习的是DOM,把文档看作对象，顶级对象是document

BOM类型事件

窗口加载事件

BOM窗口加载事件：等页面加载完毕才会触发该事件
window.onload=function(){}或者window.addEventListener(“load”,function(){})

DOM窗口加载事件:加载内容不包括图片等
document.addEventListener(“DOMContentLoad”,function(){})

调整窗口大小事件

调整窗口大小事件：窗口发生像素变化就会触发这个事件
window.onresize=function(){}或者window.addEventListener=(“resize”,function(){})

定时器

  1. window.setTimeout(调用函数，延迟的毫秒数)–到达毫秒数的时间就执行函数,window可省略
  2. window.setInterval(回调函数，间隔毫秒数)–每次到达毫秒数的时间就调用函数并且重新计时,window可省略

关闭定时器：clearTimeout/clearInterval(定时器名字)

例子练习：

  1. 发短信倒计时：使用setInterval
  2. 倒计时板块：使用setInterval
  3. 自动关闭广告：使用setTimeout

location对象

  1. location.href获取页面地址
  2. location.href=’URL’赋值以后可以跳转到别的地址
  3. location.search获取页面参数，比如你从上一个页面提交了内容，就可以在下一个页面获取提交内容
  4. location.assign()可以实现后退功能的跳转
  5. location.replace()不可以后退
  6. location.reload()刷新页面

navigator对象

navigator对象包含浏览器信息

比较常用的：navigator.userAgent–返回由客户机发送的服务器的user-agent头部值，可以判断是移动还是PC端

history对象

  1. history.back()–根据历史记录页面后退
  2. history.forward()–页面前进
  3. history.go(正数前进，负数后退)–页面前进或者后退

PC端网页特效

元素偏移量offset

  1. element.offsetParent返回该元素带有定位的父亲
  2. element.offsetTop返回该元素相对带定位的父亲上方偏移量
  3. element.offsetLeft返回该元素相对带定位的父亲左方偏移量
  4. element.offsetWidth返回padding+border+width的宽度
  5. element.offsetHeight返回padding+border+width的高度

offset和style的区别：style是行内样式，style有单位，且不包含padding和border，但是style可以改值

client

常见属性：

  1. clientTop上边框大小
  2. clientLeft左边框大小
  3. clientWidth返回自身不含边框的宽度，无单位
  4. clientHeight返回自身不含边框的高度，无单位

立即执行函数

立即执行函数：不需要调用，立马能够自己执行的函数

常见语法规范

  1. (fucntion(形参){函数语句})(实数)
  2. (fucntion(形参){函数语句}(实数))

scroll系列

  1. scrollTop返回被滚动条滚上去的内容高度距离
  2. scrollLeft返回被滚动条滚左去的内容长度距离
  3. scrollWidth返回自身实际宽度不含边框的大小
  4. scrollHeight返回自身实际高度不包含边框大小

mouseover和mouseenter的区别

区别：

  1. mouseenter只会经过自身盒子触发，不包括自己的子盒子
  2. mouseover经过子盒子也能触发
  3. 总结来说：mouseenter不会冒泡

移动端网页特效

touch触摸事件

  1. touchstart手指触碰到dom元素
  2. touchmove手指在dom元素滑动
  3. touchend手指从dom元素离开

触摸事件对象：

  1. touches所有触摸屏幕手指的列表
  2. targetTouches正在触摸当前dom元素的手指列表–重点记住
  3. changedTouches手指状态发生改变的列表，从无到有，从有到无变化

classList属性

  1. element.classList.toggle(‘类名’)切换类名，即如果有就移除，没有就添加
  2. element.classList.add(‘类名’)添加类名
  3. element.classList.remove(‘类名’)移除类名

本地存储

sessionStorage:生命周期到关闭浏览器，同一个窗口下数据共享，以赋值对的形式存储使用

  1. sessionStorage.setltem(key,value) 数据value存储给key
  2. sessionStorage.getltem(key) 获取key的数据value
  3. sessionStorage.removeltem(key) 移除key的数据
  4. sessionStorage.clear() 清除所有数据

localStorage：生命周期永久生效，除非手动删除，可以同一浏览器多窗口共享，以赋值对存储使用

  1. localStorage.setltem(key,value) 数据value存储给key
  2. localStorage.getltem(key) 获取key的数据value
  3. localStorage.removeltem(key) 移除key的数据
  4. localStorage.clear() 清除所有数据
