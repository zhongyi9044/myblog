css简介

因为HTML仅仅只在乎文本内容的含义，而不够精美，CSS就是一个美容师。CSS是一种标记语言，主要用于设置文本内容的字体大小，图片宽高，以及版面布局的外观显示样式等等等等。

css语法规范

构成：选择器+{属性：值；+属性：值：…..}
CSS需要写在head内的style(自己创建)中
例子：
<style>
p {
color: red;
font-size: 12px;
}
</style>

选择器

作用：为了将不同标签内的内容进行修改，比如上色，调节大小，我们就需要选择器来筛选出来

标签选择器

顾名思义，把一类标签名字作为选择器。

结构：
标签名 {
属性1: 属性值1;
属性2: 属性值2;
}

示例：

注意：标签选择器是把所有此标签的内容全部选择

类选择器

类选择器可以单独选择某一个标签，而不是一类标签，充分弥补了标签选择器的缺点，应用面最广

注意：类选择器的使用中，一个标签可以使用多类名

多类名的应用：如果某一些属性是需要共同拥有有的，就可以用一个类名包含这些共有的属性，再采取多类名的应用。

结构：
.自创类名 {
属性1: 属性值1;
属性2: 属性值2;
}

<div class="自创的类名+空格+自创类名+空格+....">这是一个标签选择器的例子4</div>

示例：

id选择器

id选择器也是单独选择某一标签，但是不同的是，id选择器的属性只能被选择使用一次

需要注意：id选择器没有多类名的概念

结构：
#id自创名 {
属性1: 属性值1;
属性2: 属性值2;
}

<div id="自创的id名">这是一个id选择器的例子4</div>

通配符选择器

通配符选择器是把所有的标签的内容都变成对应的属性，一般特殊情况下才使用

* {
属性1: 属性值1;
属性2: 属性值2;
}
