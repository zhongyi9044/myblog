jQuery基本使用

顶级对象

jQuery的顶级对象就是jQuery，也可以用’$’代替，相当于原生JavaScript的window

jQuery对象的本质：利用$对DOM对象包装后产生的对象，以伪数组形式存储

jQuery入口函数

入口函数的两种写法

$(function(){
...此处是页面DOM加载完成的入口
});

$(document).ready(function(){
...此处是页面DOM加载完成的入口
});

jQuery对象和DOM对象

DOM对象转换为jQuery对象：$(‘选择器’);

jQuery对象转换为DOM对象：$('选择器')[index]或者$('div').get(index)

原生的JavaScript一些方法属性jQuery没有给我们进行封装，所以需要转换才能使用

jQuery常见API

jquery基础选择器

  1. $(‘#id’) id选择器
  2. $(‘*’) 全选选择器
  3. $(‘.class’) 类选择器
  4. $(‘div’) 标签选择器
  5. $(‘div,p,li’) 并集选择器
  6. $(‘li.current’) 交集选择器
  7. $(‘ul>li’) 子代选择器，不包含孙子
  8. $(‘ul li’) 后代选择器，包含孙子

隐式迭代：给匹配到的所有元素进行循环遍历，执行相应的方法，不用我们自己循环，jQuery特有的

jQuery筛选选择器

  1. $(‘ul:first’) 获取索第一个ul元素
  2. $(‘ul:last’) 获取索最后一个ul元素
  3. $(‘ul:eq(index)’) 获取ul下该索引号元素
  4. $(‘ul:odd’) 获取ul下索引号奇数元素
  5. $(‘ul:even’) 获取ul下索引号偶数元素

jQuery筛选方法*重点

  1. $(‘li’).parent 查找父亲
  2. $(‘li’).parents(‘选择器’) 查找指定的祖先元素
  3. $(‘ul’).children(‘li’) 查找最近一级ul亲儿子li
  4. $(‘ul’).find(‘li’) 查找ul的后代li
  5. $(‘.me’).siblings() 查找当前元素兄弟节点，不包含自己
  6. $(‘.me’).nextAll() 查找当前元素之后全部同辈元素
  7. $(‘.me’).prevAll() 查找当前元素之前全部同辈元素
  8. $(‘div’).hasClass(‘protected’) 检查此元素有没有特定的类，如果有就返回true
  9. $(‘li’).eq(2) 获取li下该索引号元素

修改样式的方法

jQuery链式编程：节省代码量，更加简洁 例子：$(this).css(“color”,”red”).parent().css(“background-color”,blue)我的文字颜色变红，父亲悲剧变蓝

操作css样式方法：

  1. $(this).css(“color”) 返回属性值
  2. $(this).css(“color”,”red”) 设置样式，如果是数字可以不用单位和引号
  3. $("div").css({
            `width:100,`
        `height:100,`
        `backgroundColor:"red",`
     `})`
    第三种方法采取对象形式，方便设置样式，该方法的复合单词必须用驼峰命名法，属性之间逗号隔开

操作类样式方法

  1. $(“div”).addClass(“current”) 添加类
  2. $(“div”).removeClass(“current”) 移除类
  3. $(“div”).toggleClass(“current”) 有就移除，无就添加，即切换类

类操作和className的区别：className会覆盖其他类，类操作不会

jQuery效果

查找jQuery API：https://jquery.cuishifeng.cn/

显示和隐藏

  1. $(“div”).show(speed(速度，毫秒),easing(切换效果，快到慢等，默认swing),fn(回调函数)) 显示
  2. $(“div”).hide(speed(速度，毫秒),easing(切换效果，快到慢等，默认swing),fn(回调函数)) 隐藏
  3. $(“div”).toggle(speed(速度，毫秒),easing(切换效果，快到慢等，默认swing),fn(回调函数)) 切换

上拉下拉

  1. $(“div”).slideDown(speed(速度，毫秒),easing(切换效果，快到慢等，默认swing),fn(回调函数)) 下拉
  2. $(“div”).slideUp(speed(速度，毫秒),easing(切换效果，快到慢等，默认swing),fn(回调函数)) 上拉
  3. $(“div”).slideToggle(speed(速度，毫秒),easing(切换效果，快到慢等，默认swing),fn(回调函数)) 切换

以上显示隐藏，上拉下拉的speed，easing，fn都是可以省略不写的

事件切换

hover(over,out):mouseover和mouseout的复合写法

  1. over:鼠标移如元素要触发的函数fn
  2. out:鼠标移出元素要触发的函数fn
  3. 如果只写一个函数，那么鼠标移出移入都会触发函数fn

停止排队

stop():用于停止动画或者效果，一般用于动画和效果前面，使上一个动画或者效果停止

使用举例：$(“div”).stop().slideToggle();

淡入淡出效果

  1. $(“div”).fadeIn(speed(速度，毫秒),easing(切换效果，快到慢等，默认swing),fn(回调函数)) 淡入，逐渐出现
  2. $(“div”).fadeOut(speed(速度，毫秒),easing(切换效果，快到慢等，默认swing),fn(回调函数)) 淡出，逐渐消失
  3. $(“div”).fadeToggle(speed(速度，毫秒),easing(切换效果，快到慢等，默认swing),fn(回调函数)) 淡入淡出切换
  4. $(“div”).fadeTo(speed(速度，毫秒，此方法中必写),opacity(透明度，0-1之间，必写),easing(切换效果，快到慢等，默认swing),fn(回调函数)) 调整透明度

自定义动画

animate(params,speed,easing,fn):params是必须要写的，采用对象的形式书写，复合型用驼峰命名法

使用举例：
$("div").animate({
left:100,
top:100,
backgroundColor:"red",
});
以动画的效果将样式变成举例左100px，上100ox，背景变红

jQuery属性操作

  1. prop(“属性”) 获取属性值
  2. prop(“属性”,”属性值”) 设置属性值
  3. attr(“属性”) 获取自定义属性值
  4. arrt(“属性”,”属性值”) 设置自定义属性值

jQuery内容文本值

  1. html() 获取元素内容
  2. html(“内容”) 设置元素内容
  3. text() 获取元素文本内容
  4. text(“内容”) 设置元素文本内容
  5. val() 获取表单值
  6. val(“内容”) 设置表单值
  7. toFixed(n) 保留n位小数

jQuery元素操作

遍历元素each:一般使用在需要将投以雷元素进行同类不同样操作

  1. $(“div”).each(function(index,domEle){}–index是下标，自动给你分配了；domEle是Dom对象，在function要使用jQuery方法需要转换为jQuery对象
  2. $.each(object,function(index,element){})–object可以是任何对象，数组；index是下标，自动给你分配了；element是遍历内容值

创建 删除 添加元素

  1. var li=$(<li>我是li</li>) 创建了一个li，且内容是‘我是li’
  2. $(“ul”).append(li) 在ul内部最后面添加了li，等于最后一个儿子
  3. $(“ul”).prepend(li) 在ul内部最前面添加了li，等于第一个儿子
  4. $(“ul”).after(li) 在ul外部后面添加了li，等于兄弟
  5. $(“ul”).before(li) 在ul外部前面添加了li，等于兄弟
  6. $(“ul”).remove() 删除了ul
  7. $(“ul”).empty() 删除了ul的所有子代
  8. $(“ul”).html(“”) 删除了ul的所有子代，””是不可以省略的

jQuery尺寸位置操作

尺寸操作：返回的是数字型

  1. width()/height() 获取元素宽/高，如果需要设置尺寸，在括号里输入数字不加单位
  2. innerWidth()/innerheight() 获取元素宽/高，包含padding
  3. outerWidtn()/outerheight() 获取元素宽/高，包含padding，border
  4. outerWidtn(true)/outerheight(true) 获取元素宽/高，包含padding，border，margin

位置操作：

  1. offset() 获取距离文档顶部和左边距离，此方法有且仅有两个属性，top和left
  2. offset({top:xx,left:xx}) 设置距离文档顶部和左边距离，不需要单位
  3. position() 获取相对于带有定位的父亲的偏移坐标，此方法只能读取，不能设置偏移
  4. scrollTop()/scrollLeft() 获取所取元素被卷去上部和左部距离
  5. scrollTop(xx)/scrollLeft(xx) 使所取元素到达恰好被卷去上部或左部的距离的位置

jQuery事件

单个事件注册：$(“div”).click(function(){})

on()绑定事件：可以一次绑定多个事件

  1. $(“div”).on({click:function(){},mouseover:function(){}}) div绑定了click，mouseover事件
  2. $(“div”).on({“click mouseover”,function(){}}) div绑定了click，mouseover事件，且恰好两个事件处理程序一样

on()绑定事件，实现事件委托：将子元素身上的事件绑定在父元素身上

  1. $(“ul”).on({“click”,”li”,function(){}})

优势：可以动态的进行事件绑定，不再需要创造一个li就绑定一个事件，并且比传统的事件委派简介

off()解绑事件

  1. $(“div”).off() 解除所有事件
  2. $(“div”).off(“click”) 解除click事件
  3. $(“div”).off(“click”,”li”) 解除事件委托

one()方法：$(“div”).one(“click”,function(){}) 只触发一次的事件

自动触发事件：

  1. $(“div”).click() 传统方式
  2. $(“div”).trigger(“click”) jQuery方式
  3. $(“div”).triggerHandler(“click”) 和trigger一样，但是此方法不会触发元素的默认行为，比如事件触发以后自动产生光标闪烁不会触发

jQuery事件对象：function(event)

  1. event.preventDefault() 阻止默认行为
  2. event.stopPropagation() 阻止冒泡

jQuery事件拷贝：$.extend(deep,target,object,objects)

  1. deep默认使false，浅拷贝，true为深拷贝
  2. target为被拷贝复制的目标
  3. object为拷贝粘贴目标
  4. objects为第二个，第三个，第四个。。。。拷贝粘贴目标
  5. 浅拷贝和深拷贝的区别在于，复杂数据类型在浅拷贝的时候拷贝给的是地址，所以修改object拷贝来的复杂数据类型的值进行修改，原来的target的值也会被修改，深拷贝则不会

复杂数据类型例子：mgs：{age：18，name：”john”}
普通数据类型例子：id：10
